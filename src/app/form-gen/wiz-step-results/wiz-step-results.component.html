Step 3: Generated designer
Important:

Before going artistic with your form, you must perform the following actions:

  • Open with Visual Studio, the project where your form is located
  • Close the form design if open
  • Open the form.designer.cs
  • Replace all content from your form.designer.cs file with the text below
  • Change the designer.cs namespace
  • Do not worry about spaces and wrong order of some elements. They will be resolved in the following steps
  • Save the file
  • Open the form design
  • Enable 'Localizable' and 'Text' from the property editor
  • Reopen form.designer.cs and format the document (either press ctrl+K D, or select Edit->Advanced->Format document from menu
  • Save the form.designer.cs file
  • Your form is ready

The generated code container the assigned controls for each property you selected, with its datasource bindings and layout captions.
